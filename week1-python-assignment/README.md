# Week 1 Python Assignment

This repository contains clean, beginner-friendly Python programs completed as part of the Week 1 Python Internship Assignment. 

These programs are written in plain, modular Python 3, making them easy to understand, test, and explain during a technical interview.

---

## Project Structure and Files

The project contains the following files:

1. **`temperature_converter.py`**
   - A command-line program that converts temperature values between Celsius and Fahrenheit.
   - Includes basic input error handling to prevent program crashes on invalid user entries.

2. **`grade_calculator.py`**
   - Calculates a student's total, average, percentage, and assigned letter grade (A, B, C, D, or F).
   - Validates that marks are within the correct range (`0 to 100`).

3. **`prime_even_odd.py`**
   - Evaluates whether an integer is even or odd, and checks if it is a prime number.
   - Accurately handles edge cases such as negative numbers, zero, and one.

4. **`atm_simulator.py`**
   - A menu-driven ATM console simulation.
   - Features: PIN Login verification (Default PIN: `1234`), balance checks, deposit processing, withdrawal boundaries, and PIN change mechanics.

5. **`practice_questions.py`**
   - An interactive collection of 5 common interview algorithms:
     1. Area of a Circle
     2. Even/Odd parity check
     3. Factorial using recursion
     4. Vowel and consonant counting
     5. Fibonacci series generator
